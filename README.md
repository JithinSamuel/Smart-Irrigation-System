# 🌱 Smart Irrigation System

An IoT-based ESP32 smart irrigation system that automates plant watering based on soil moisture. It shows real-time data on OLED and LCD displays, updates Blynk and ThingSpeak platforms, and uses RGB LED and an active buzzer module for alerts.

---

## ✨ Key Features

- **Smart Watering**: Automatic pump control based on soil moisture  
- **Environmental Monitoring**: Temperature, humidity, and water level sensing  
- **Real-time Display**: OLED and LCD show live sensor data  
- **IoT Connectivity**: Remote monitoring via Blynk and ThingSpeak  
- **Alert System**: RGB LED + Active Buzzer Module for status and warnings  

---

## 🧰 Components Used

- ESP32 DevKit  
- DHT11 Sensor  
- Capacitive Soil Moisture Sensor  
- Water Level Sensor  
- Relay Module + 12V Mini Pump  
- OLED Display (0.96")  
- LCD (I2C)  
- RGB LED  
- Active Buzzer Module
- Breadboard + Power Supply Module  
- Jumper Wires
